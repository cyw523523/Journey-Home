1. 用户模块

● 用户注册、登录、退出登录

● 个人信息查看与修改

● 游客模式

● 个人信息/救助站实名

● 查看本人发布的档案、救助、领养记录

● 权限区分：普通用户 / 管理员

2. 流浪猫狗档案模块

● 新增动物档案（基础信息 + 照片上传）

● 编辑、删除本人发布的档案

● 查看档案列表与详情

● 多条件筛选：宠物类型、性别、状态、发现地区

● 状态管理：待救助 / 救助中 / 待领养 / 已领养

3. 救助信息模块

● 发布救助信息（关联对应动物档案）

● 编辑、删除本人发布的救助信息

● 查看救助信息列表与详情

● 更新救助状态：待处理 / 处理中 / 已完成

4. 领养信息模块

● 发布领养信息

● 提交领养申请（关联目标动物）

● 查看本人领养申请进度与结果

5. 管理员后台模块

● 管理员登录后台

● 用户管理（查看用户列表、禁用违规用户）

● 动物档案审核、救助信息审核、领养申请审核

● 公告发布与管理

● 基础数据统计（动物总数、救助/领养数量、状态分布）

6. 公共展示模块

● 首页展示最新救助/领养信息

● 动物档案列表展示

● 分类检索与关键词搜索

● 系统公告展示
